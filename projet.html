Triangle de Sierpinski

Sur cette page, vous allez trouver le projet sur python que nous devions faire, le triangle de Sierpinski (ci-dessous)

Ce projet était un peu complexe, pour être honnête les deux derniers jours des vacances, j'ai pris la plupart de mon temps pour faire ce travail. Le programme est codé entièrement sur Python.

Le projet consiste à automatiser à l'aide du module turtle, un programme permettant de réaliser la figure ci contre, le triangle de Sierpinski. Le programme est annoté et donc la description du programme est dans le code directement.
